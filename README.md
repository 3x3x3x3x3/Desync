# Desync

**Desync** is a universal Roblox desync script that requires an executor with the `setfflag` function.  

## Tested Games
- Bedwars  
- Baseplate  

## Supported PC Executors
- Volcano  
- Seliware  
- Wave  
- Potassium  

## Notes
- Make sure your executor supports `setfflag` before using.  
- Use at your own risk
